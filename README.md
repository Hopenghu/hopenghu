# HopenGhu Website Project

## 開發環境設置

### 必要條件
- Node.js
- Python 3
- Git

### 安裝步驟
1. 克隆專案
```bash
git clone [your-repository-url]
cd hopenghu
```

2. 安裝依賴
```bash
npm install
```

3. 啟動開發服務器
```bash
npm start
# 或使用
npm run dev
```

### 專案結構
```
hopenghu/
├── public/
│   ├── css/
│   ├── images/
│   └── js/
├── pages/
│   ├── index.html
│   ├── explore.html
│   └── ...
└── src/
    ├── js/
    └── css/
```

## 部署
使用 Cloudflare Pages 進行部署。

## 開發指南
1. 使用 `npm run format` 格式化代碼
2. 遵循 HTML5 和 CSS3 標準
3. 確保響應式設計支持
